# P-Ezy – Smart Traffic & Fine Management System

P-Ezy is a traffic fine management system designed to digitize traffic violation handling in Sri Lanka.  
The solution includes:

- **Police Officer Mobile Application** (field use)
- **Driver Web Portal** (view/pay fines online)
- **Admin Web Dashboard** (monitoring, reporting, user management)

## 🚦 Why P-Ezy?
The current traffic fine process is largely manual and paper-based, causing delays, limited real-time access to records, and high administrative workload. P-Ezy introduces a centralized digital platform to improve efficiency, transparency, and accountability.

## 👥 User Roles & Platforms

### 👮 Police Officers (Mobile Application)
Used during roadside checks via mobile phone:
- Traffic fining and checking
- Traffic violation history checking
- Criminal record list (Red notice / Most wanted list)

### 🚗 Drivers (Web Portal)
- View issued fines and violation history
- Pay fines online securely
- Download receipts and track payment status

### 🛡️ Administrators (Web Dashboard)
- User and role management (RBAC)
- Fine/payment monitoring and audit logs
- Reports & analytics (violation trends, payments, officer activity)

## ✨ Key Features
- Digital fine issuance (Police mobile app)
- Secure online payments (Driver portal)
- Real-time record verification (Police)
- Role-based access control (Drivers / Police / Admin)
- Centralized database for fines, payments, and records
- Notifications and reminders
- Reporting and analytics dashboards
